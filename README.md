# giphy-explorer
Android app using Giphy public API

App is built using:
 - MVVM architecture
 - Hilt
 - Retrofit + GSON 
 - Kotlin
 - Coroutines
 - Compose
 - paging-compose
 - Turbine and Junit for unit tests
